Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table Create list using tr, th, rd
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header container for introductory content
div divider container for content
ol ordered list numbered list
ul unordered list bullet point list
li list item put items in ol or ul
blockquote blockquote enclosed text is an extended quote

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong bold text <em>strong</em> strong
a anchor link text <a href="http://example.com"> click here
img image input image for website <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code Monospace font <code>example of text</code> woohoo
span span Group inline-elements <span>span example</span> this is a span

Document Tags

Tag Purpose Example
html convention to add html tag <html>insert body here</html>
head container for all head elements <head>insert header elements here</head>
body container for document's body <body>insert main content here</body>
body define character sets <charset="utf-8">
title set tab title in browser <title>insert title here</title>
